# PiConsole

> A handbuilt retro gaming console designed from scratch by 2 game enthusiasts.

![PiConsole](https://github.com/Piconsole/PiConsole/blob/main/layout.webp)
> Fig: Rough layout of the console (No 3D version yet lol)

## What is PiConsole?

PiConsole is a handheld gaming console built from custom components, meaning no kits or any shortcuts.
Every component was choosen deliberately and everything matters here.

The goal is to understand how hardware and software come together to make what each one of us has
enjoyed doing as kids, like playing games etc.

## Parts we used

- Raspberry Pi Zero 2W (the brain)
- 3.2" inch Screen
- Buttons via MCP23017 GPIO Expander
- 3000 mAh rechargeable battery
- Audio Module
- 32 GB microSD (class 10)
- Charger Module
- MT3608 Boost Converter
- AMP Module
- Wires
- Power Switch

- Custom 3D printed design
